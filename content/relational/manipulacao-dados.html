Manipulação de Dados (DML)

Aprenda a inserir, alterar e remover registros do banco de dados com segurança.

1. Inserindo Dados (INSERT)

O comando de>INSERT INTO é usado para adicionar novas linhas.

de>INSERT INTO PERSONS (ID, NAME, COUNTRY, JOB)
  VALUES (10, 'Santos Dumont', 'Brazil', 'E');
Dica: Se você listar as colunas explicitamente, não precisa fornecer valores para todas (as outras ficarão como NULL ou valor padrão).

2. Atualizando Dados (UPDATE)

Use para modificar registros existentes.

de>UPDATE PERSONS
  SET JOB = 'S'
  WHERE NAME = 'Santos Dumont';
PERIGO: Sempre use a cláusula de>WHERE. Se esquecer, todas as linhas da tabela serão alteradas!

3. Removendo Dados (DELETE)

Use para apagar registros.

de>DELETE FROM PERSONS
  WHERE ID = 10;
