Projetos Práticos NoSQL

Ideias para portfólio usando tecnologias não relacionais.

1. Crawler de Redes Sociais (MongoDB)

Tecnologia: Document Store (MongoDB).

Projeto: Script Python que baixa tweets ou posts públicos e salva em JSON bruto.

Desafio: Lidar com a estrutura variada (alguns posts tem foto, outros localização, outros apenas texto). O NoSQL aceita tudo sem reclamar.

2. Chat em Tempo Real (Redis)

Tecnologia: Key-Value (Redis).

Projeto: Um sistema simples de chat.

Desafio: Usar a estrutura de listas do Redis para guardar as últimas 50 mensagens em memória RAM para acesso instantâneo (sub-milissegundo).

3. Sensores IoT (Time Series)

Tecnologia: InfluxDB ou Timescale.

Projeto: Simular 1000 sensores de temperatura enviando dados a cada segundo.

Desafio: Agregação temporal (média de temperatura por hora) em tempo real.
