Definição de Dados (DDL)

Criando a estrutura do banco: Tabelas, Chaves e Visões.

1. Criando Tabelas (CREATE TABLE)

Define o nome da tabela e o tipo de cada coluna.

de>CREATE TABLE PRODUTOS (
      ID INT PRIMARY KEY,
      NOME VARCHAR(100) NOT NULL,
      PRECO DECIMAL(10, 2),
      CRIADO_EM DATE
  );

2. Restrições (Constraints)

ConstraintDescrição
PRIMARY KEYIdentificador único. Não aceita nulos.
FOREIGN KEYGarante que o valor existe em outra tabela (relação).
UNIQUEImpede valores duplicados na coluna.

3. Views (Visões)

Uma "tabela virtual" baseada em uma consulta SELECT.

de>CREATE VIEW CLIENTES_VIP AS
  SELECT * FROM CLIENTES WHERE GASTO > 1000;
