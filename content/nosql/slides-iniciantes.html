Slides: O Mundo NoSQL

Apresentação sobre a mudança de paradigma: de dados estruturados para flexíveis.

O Cenário Moderno (Big Data)

Até os anos 2000, a maioria dos dados eram transacionais (Bancos, Vendas).

Hoje, lidamos com os 3 V's:

Volume

Dados demais para um único servidor.

Velocidade

Dados chegando em milissegundos.

Variedade

Texto, áudio, vídeo, JSON.

Notas: Enfatize a "Variedade". SQL sofre para guardar um tweet, um log de servidor e uma foto na mesma tabela. NoSQL facilita isso.

O Problema da Escala

SQL (Vertical) NoSQL (Horizontal)
Para aguentar mais dados, preciso de um computador mais potente (Mais CPU/RAM). Para aguentar mais dados, adiciono mais computadores simples (Clusters).
Custo cresce exponencialmente. Custo cresce linearmente.

NoSQL nasceu para rodar em clusters distribuídos.
