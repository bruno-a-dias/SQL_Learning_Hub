SQL Intermediário: Operadores Lógicos e Agregações

Avance além do básico. Aprenda a combinar múltiplas condições, buscar padrões de texto e calcular estatísticas sobre seus dados.

1. Combinando Condições (AND, OR, NOT)

No mundo real, filtros raramente são simples. Frequentemente precisamos de perguntas compostas, como "Quero clientes do Brasil E que sejam maiores de idade".

AND (E)

Exige que todas as condições sejam verdadeiras.

de class="code-small">WHERE COUNTRY = 'USA' AND JOB = 'S'

OR (OU)

Exige que pelo menos uma condição seja verdadeira.

de class="code-small">WHERE COUNTRY = 'USA' OR COUNTRY = 'Brazil'

NOT (NÃO)

Inverte o resultado. Seleciona o que não atende à condição.

de class="code-small">WHERE NOT COUNTRY = 'UK'

2. Operadores Especiais de Filtragem

SQL oferece atalhos poderosos para tornar o código mais limpo e legível.

O Operador IN

Substitui múltiplos de>OR. Verifica se o valor está dentro de uma lista.

de>SELECT * FROM PERSONS WHERE COUNTRY IN ('USA', 'UK', 'France');

O Operador BETWEEN

Seleciona valores dentro de um intervalo (inclusive os extremos). Funciona para números e datas.

de>SELECT * FROM COUNTRIES WHERE LITERACY BETWEEN 90 AND 100;

O Operador LIKE (Busca de Padrões)

Usado para buscar partes de um texto. Utiliza "curingas":

  • % (Porcentagem): Substitui zero ou mais caracteres.
  • _ (Sublinhado): Substitui exatamente um caractere.

Exemplo: Encontrar nomes que começam com a letra "A".

de>SELECT NAME FROM PERSONS WHERE NAME LIKE 'A%';

3. O Conceito de NULL

de>NULL em banco de dados não é zero, nem espaço em branco. NULL significa ausência de informação (desconhecido).

Cuidado: Você nunca deve usar de>= NULL. A forma correta de verificar é usando os operadores:

  • de>IS NULL (É nulo)
  • de>IS NOT NULL (Não é nulo)

4. Funções de Agregação (Estatística)

Até agora olhamos linha por linha. As funções de agregação permitem "resumir" os dados, calculando totais e médias sobre um grupo de registros.

Função Descrição Exemplo
de>COUNT(*) Conta quantas linhas existem. de>SELECT COUNT(*) FROM PERSONS;
de>SUM(coluna) Soma os valores de uma coluna numérica. de>SELECT SUM(POP) FROM COUNTRIES;
de>AVG(coluna) Calcula a média aritmética. de>SELECT AVG(LITERACY) FROM COUNTRIES;
de>MAX(coluna) Encontra o maior valor. de>SELECT MAX(AREA) FROM COUNTRIES;
de>MIN(coluna) Encontra o menor valor. de>SELECT MIN(BDATE) FROM PERSONS;
