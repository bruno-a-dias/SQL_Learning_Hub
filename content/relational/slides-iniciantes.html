Fundamentos de SQL

Módulo Iniciante

Tópicos desta aula:

  • O conceito de Banco de Dados Relacional
  • Tabelas, Linhas e Colunas
  • Comando SELECT e FROM
  • Filtragem básica com WHERE
Notas: Comece explicando que SQL é uma linguagem padrão. Ressalte que não é uma linguagem de programação comum (como Python ou Java), mas uma linguagem de consulta declarativa.

Anatomia de uma Tabela

Uma tabela é como uma planilha rigorosa:

Coluna (Atributo) Coluna (Atributo)
Dado (Linha 1) Dado (Linha 1)
Dado (Linha 2) Dado (Linha 2)

Regra de Ouro: Cada coluna tem um tipo de dado fixo (só números, só texto, só datas).

Notas: Use a analogia da planilha do Excel, mas enfatize a tipagem forte. Uma coluna de 'Idade' não aceita escrever 'Vinte anos' por extenso se for numérica.

A Frase SQL Básica

SELECT colunas
FROM tabela
WHERE condição;

A ordem importa! O computador lê: "Selecione X, da tabela Y, onde Z é verdade".
